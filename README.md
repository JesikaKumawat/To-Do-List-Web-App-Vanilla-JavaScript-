# To-Do List Web App  
A simple and clean To-Do List application built using **HTML**, **CSS**, and **Vanilla JavaScript**.  
This project helps users add, delete, and mark tasks as completed — perfect for beginners learning web development.

---

## Features

- Add new tasks  
- Mark tasks as completed  
-  Delete tasks  
-  Simple and clean UI  
-  100% Vanilla JavaScript (no frameworks)

---

##  How to Run

1. Clone or download this repository  
2. Open the folder in **VS Code**  
3. Open `index.html` in your browser  
   - (Optional) If using VS Code, right-click → **Open With Live Server**

---

##  How It Works

### 1. Add Tasks  
Type a task in the input field and click the **Add** button.

### 2. Mark as Completed  
Click on the task text to toggle the **completed** state.

### 3. Delete Tasks  
Click the **X** button to delete any task.
